# Arduino Nano программатор микросхем EPROM/Flash 27,28,29 серий, тестер RAM и микросхем логики

Данное устройство распространяется как есть, автор не несет ответственность за порчу вашего имущества, все действия выполняются исключительно на ваш страх и риск!

Важно:
 * Программатор поддерживает только микросхемы с питанием 5 вольт.
 * Необходимо настроить в скетче и проверить правильность показаний вольтметра в программе.
 * Алгоритмы работы программатора могут не полностью соответствовать даташитам (краткое описание алгоритмов программатора в файле Algorithm.txt).
 * Панельки ZIF32 и ZIF16 используются по отдельности, запрещено одновременное нахождение микросхем в обоих панельках сразу.
 * Для EPROM/Flash и StaticRAM используется панелька ZIF32 (первая ножка микросхемы располагается в сторону рычажка ZIF-панели).
 * Для Dynamic RAM используется панелька ZIF16 (ОБРАТИТЕ ВНИМАНИЕ - первая ножка микросхемы располагается в противоположную сторону от рычажка ZIF-панели).
 * Для микросхем логики используется панелька ZIF16 (первая ножка микросхемы располагается в сторону рычажка ZIF-панели).
 * Микросхемы, которые имеют меньшее количество выводов, при установке должны выравниваться по противоположной стороне от рычажка ZIF-панели.
 * Уточнить (по картинке) расположение микросхемы в панельке можно, нажав кнопку "Chip Info".
 * На плате присутствуют JP1 и JP2, программа напоминает об установке джамперов, определитесь для себя в каком положении должны быть джамперы на вашей плате (согласно схемы). 
 * Перед установкой микросхемы в панельку программатора необходимо в программе выбрать нужный тип микросхемы.
 * Убедитесь, что расположение микросхемы в панельке правильное.
 * Убедитесь в надежном контакте всех ножек микросхемы и контактной площадки.
 * Перед использованием той или иной функции уточняйте в даташите возможность поддержки чипом этой функции, особенно где используется высокое напряжение.
 * Перед использованием режимов Write/Erase/Autoselect, уточните необходимое высокое напряжение для вашей микросхемы в datasheet'е!
 * Ручная настройка высокого напряжения для режимов Write/Erase/Autoselect.
 * Представленная в программе информация несет только ознакомительный характер.
 * На плате присутствует разъем для подключения дополнительного питания Ардуино (7-12В, pin Vin), для случаев если питание от USB будет недостаточным для некоторых микросхем.
 * Все протестированные микросхемы работали от USB и напряжении питания на выводе Ардуино 5V = 4.6 вольт.
 * Если программатор не сможет протестировать или корректно работать с микросхемой это не означает, что чип точно нерабочий.

Функции:
 * Чтение чипа (с проверкой на пустоту FF).
 * Запись чипа:
    Для чипов 27С серии - с выбором времени Chip Enable Program Pulse Width (с подачей напряжения согласно даташиту на pin VPP).
    Для чипов 28С и 29С серии - с выбором количества байт Page Write (согласно даташиту).
    Для чипов 28F серии - с подачей напряжения согласно даташиту на pin VPP.
    Для чипов 28С, 29С, 29F серий перед записью ячейки стираются автоматически без высокого напряжения.
    Для чипов 28F серии перед записью обязательно нужно стереть чип.	
 * Верификация и проверка чипа после записи.
 * Стирание электрически стираемых чипов.
    Для чипов 27С серии - с подачей напряжения согласно даташиту на pins A9 и VPP (доступно только после "Autoselect" и для чипов, для которых указано напряжение стирания). 
    Для чипов 28С серий - с подачей напряжения согласно даташиту на pin OE.
    Для чипов 28F серий - с подачей напряжения согласно даташиту на pin VPP.
 * Soft стирание (без подачи высокого напряжения) для чипов 28С, 29С, 29F серий.
 * Чтение сигнатур (Autoselect) для чипов 27С, 28F, 29C, 29F серий (с подачей 12в на pin A9).
    При плохом контакте может считать неверные сигнатуры, которые могу совпасть с сигнатурами других чипов.		
 * Чтение и запись Device Identification для чипов 28С серии (с подачей 12в на pin A9).
 * Включение-Отключение Software Data Protection (функция защиты записи данных) для некоторых чипов 28С, 29С серий.
    Для чипов без установленной защиты выбирать SDP Disable (для отключения) не нужно. 
 * Поддержка модульности для некоторых чипов 28С серии.
 * Тестирование RAM и Logic микросхем.
 * Отображение краткой информации о чипе.
 * Сохранение считанного кода прошивки в файл.

Список поддерживаемых микросхем (не все микросхемы протестированы):
 Для EPROM/Flash и SRAM - DIP24,28,32 (панелька ZIF32), для DRAM - DIP16 (панелька ZIF16), для Logic - DIP14,16 (панелька ZIF16)
 Алгоритм 27Cxxx:
 * 27C16
 * 27C32
 * 27C64
 * 27C128 (протестировано чтение, определение сигнатуры)
 * 27C256 (протестировано чтение, определение сигнатуры)
 * 27C512 (протестировано чтение, запись, стирание, определение сигнатуры)
 * 27C010
 * 27C020
 * 27C040
 * 27C080
 Алгоритм 28Cxxx:
 * 28C04
 * 28C16
 * 28C17
 * 28C64
 * 28C256
 * 28C512
 * 28C010
 * 28C020
 * 28C040
 Алгоритм 28Fxxx:
 * 28F526
 * 28F512
 * 28F010
 * 28F020
 Алгоритм 29Cxxx:
 * 29C256
 * 29C512
 * 29C010
 * 29C020 (протестировано чтение, запись, стирание, определение сигнатуры, Software Data Protection)
 * 29C040
 Алгоритм 29Fxxx:
 * 29F512
 * 29F010
 * 29F020
 * 29F040 
 Алгоритм SRAM:
 * SRAM 16K (2K x 8-Bit)
 * SRAM 32K (4K x 8-Bit)
 * SRAM 64K (8K x 8-Bit) (протестировано)
 * SRAM 128K (16K x 8-Bit)
 * SRAM 256K (32K x 8-Bit) (протестировано)
 * SRAM 512K (64K x 8-Bit)
 * SRAM 1024K (128K x 8-Bit)
 * SRAM 2048K (256K x 8-Bit)
 Алгоритм DRAM:
 * 4164
 * 41256 (протестировано)
 Logic tester:
 * Микросхемы логики серий 4ххх, 74ххх (смотрите в файле ListLogic.txt)
   Протестировано (К176ЛА7 (CD4011), HCF4016BE, CD4028BE, HCF4040BE, HCF4049UBE, CD4066BE, CD4070BCN, SN74ALS05AN, SN74HC595N)

Переключатель JP1:
 Положение 0-1 подача VCC для DIP24
 Положение 0-2 подача VCC для DIP28
 Положение 0-3 подача VCC для DIP32

Переключатель JP2:
 Положение 0-1 подача VPP для 27C16
 Положение 0-2 подача VPP для 27C64, 27C128, 27C256
 Положение 0-3 подача VPP для 27C32, 27C512, 27C080; напряжения для стирания 28Cxxx
 Положение 0-4 подача VPP для 27C010, 27C020, 27C040, 28Fxxx; напряжения для стирания 28Fxxx
